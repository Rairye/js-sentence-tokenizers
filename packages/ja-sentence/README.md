## Installation

1. Download the tgz file
2. In the project directory, excute:

npm install ja-sentence-1.0.0.tgz


## Code Sample

```javascript

import tokenizeJapaneseParagraph from 'ja-sentence';

var results = tokenizeJapaneseParagraph("えー！？くれるの？本当にいいの…？嬉しい！！");

console.log(results);

```
